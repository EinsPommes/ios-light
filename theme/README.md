# iOS Light

A beautiful iOS-inspired Discord theme featuring San Francisco fonts, iOS color palette, rounded corners, blur effects, and modern UI elements.

## Installation

### Vencord/BetterDiscord (or any client that supports theme files)

1. Download the theme file: [`ios-light.theme.css`](https://github.com/EinsPommes/ios-light/blob/main/theme/ios-light.theme.css)
2. Drag the file into your theme folder
3. Enable the theme in your client settings

### Install through link

Add `https://einspommes.github.io/ios-light/build/ios-light.css` to your theme import links.
